# アイコンテンプレート

このディレクトリに PNG 画像を置くと、クローラー起動時に自動読み込みされ
テンプレートマッチングによるアイコン検出に使用される。

## ファイル命名規則

```
{icon_name}.png
```

ファイル名（拡張子なし）が検出結果の `text` フィールドの `icon:` プレフィックスに続く文字列になる。

例:
- `close_btn.png` → `"text": "icon:close_btn"`
- `menu_btn.png`  → `"text": "icon:menu_btn"`
- `back_arrow.png` → `"text": "icon:back_arrow"`

## テンプレート作成の指針

1. **解像度**: 実機スクリーンショットからアイコンを切り抜いてそのまま使用する
   - iOS @3x: 実際のピクセルサイズをそのまま保存
   - Android: デバイス解像度に合わせる
2. **余白**: アイコン本体の周囲 2〜4px のみ。余白が多いと誤検出が増える
3. **背景**: 可能な限り背景が一定のアイコンを選ぶ（透過 PNG 不可 → グレースケール）
4. **信頼度閾値**: デフォルト 0.80。テンプレートを追加したら実際のスクリーンで
   `CrawlerConfig.icon_threshold` を調整する

## 推奨テンプレート例

| ファイル名 | 対象 | 備考 |
|-----------|------|------|
| `close_btn.png` | × 閉じるボタン | 右上隅に多い |
| `menu_btn.png` | ☰ ハンバーガーメニュー | 左上隅に多い |
| `back_arrow.png` | ← 戻る矢印 | iOS/Android 共通 |
| `settings_gear.png` | ⚙ 設定歯車 | 右上に多い |
