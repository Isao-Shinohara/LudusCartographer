### Phase 7: OpenCV ハイブリッド検出 (OCR + Template Matching)

#### 設計決定
1. **Confidence 設計**: `cv2.matchTemplate(TM_CCOEFF_NORMED)` スコア [0.80, 1.00] を採用。
   OCR の `confidence` フィールドと同一フォーマットで保存。
2. **NMS**: IoU ≥ 0.4 をスコア降順でフィルタ → 同一アイコン重複を除去、別位置アイコンを保持。
3. **`ocr_results` 非統合**: 指紋計算 (`_generate_fingerprint`) とタイトル抽出 (`_extract_title`)
   への汚染を防ぐため、アイコン検出結果は `tappable_items` に直接追加。
4. **Graceful Degradation**: テンプレートなし→`[]`、ImportError→`[]`、無効パス→`[]`。
