## 発生した問題と解決

| 問題 | 原因 | 解決 |
|------|------|------|
| SSH push 失敗 | SSHキーが別アカウント (isaoshinohara) に紐付き | HTTPS + gh token で push |
| Appium ERR_REQUIRE_ESM | Node.js v21 が非互換 | nodebrew で v18.20.8 LTS に切り替え |
| xcuitest インストール失敗 | バージョン指定が必要 | `appium driver install xcuitest@8` |
| PaddleOCR `use_gpu` エラー | 3.x API 変更 | `device="cpu"` に変更 |
| PaddleOCR 出力解析エラー | 3.x で `rec_texts/rec_scores/rec_polys` に変更 | `predict()` + 新キー構造に対応 |
| Python 3.9 `str\|Path` 型エラー | 3.10以降の構文 | `Union[str, Path]` に変更 |
